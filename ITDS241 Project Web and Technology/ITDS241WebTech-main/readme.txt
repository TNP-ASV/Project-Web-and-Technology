#How to use 73GANE Cars Website
เป็นเว็ปไซต์สำหรับการขายรถยนต์นำเข้าของบริษัท 73GANE Cars 

##สารบัญ Table of Contents
-ขั้นตอนการติดตั้ง
-วิธีการรัน
-วิธีการใช้งาน Website


##ขั้นตอนการติดตั้ง
ดาวน์โหลดไฟล์ Zip ที่ให้ไปและทำการแตกไฟล์ ตรวจเช็คข้อมูลเบื่องต้นให้เรียบร้อยว่าดาวน์โหลดครบทั้งหมด ประกอบไปด้วย 4 Folder และ 1 File ได้แก่  css , images , js , pages และ File readme.txt

##วิธีการรัน
วิธีการรันตัว Website สามารถรันได้ 2 วิธี
1.รันโดยการเปิด Browser โดยตรง
2.รันโดยการเปิดใน Code Editer เช่น Visual Studio Code
 ที่มีการติดตั้ง Live Sever 


##วิธีการใช้งาน Website
Website มีทั้งหมด 7 หน้า โดยแต่ละหน้าจะสามารถลิ้งค์ไปหน้าอื่นๆได้ตาม Navigational Diagram ที่แนบมา ในแต่ละหน้าจะมีจุดประสงค์และรายละเอียดในการใช้งานดังนี้
1.HomePage
- เป็นหน้าเริ่มต้นของ Website จะมีรายละเอียดเกี่ยวกับบริษัท 73GANE Cars และเป็นช่องทางในการประชาสัมพันธ์สินค้าและโฆษณาต่างๆ เช่น สินค้ามาใหม่ที่พร้อมแนบรูปกับราคาสินค้านั้นๆ
2.Log-in Page
- เป็นหน้าที่ใช้สำหรับการยืนยันตัวตนของผู้เข้าใช้งานและส่วนของแอดมิน
- ในส่วนของแอดมินเพจนั้นมีไว้เพื่อที่จะสามารถเข้าไปจัดการสินค้า และ Account ของแอดมินได้โดยการกดปุ่ม Log in เพื่อเข้าหน้า Admin Page
3.Search Page
- หน้านี้จะใช้ในการ Search หาสินค้า โดยการ Search หานั้นจะทำได้ 2 รูปแบบ
> นั้นคือการ Search โดยใช้ Keyword เช่น ชื่อรุ่น ,  ชื่อโมเดล , ชื่อยี่ห้อ เป็นต้น
> เป็นการเลือก Car Type เพื่อใช้แสดงผลการค้นหาว่าผู้เข้าใช้มีความต้องการเลือกซื้อแบบประเภทใด
4.ABOUTUS
- จะมีข้อมูลผู้จัดทำ Website
- ส่วนของ Footer จะมีข้อมูลที่ตั้ง Showroom , Service ต่างๆ และ Contact สำหรับช่องทางในการติดต่อ
5.ProductDetail
- ใช้สำหรับจำหน่ายและแสดงข้อมูลรายละเอียดสินค้า เช่น รูปภาพรถ , ชื่อยี่ห้อ , ชื่อรุ่น , ราคา , ปีรถ , เครื่องยนต์ , ระบบส่งกำลัง , สีรถ
6.Admin Page
- เป็นส่วนของหน้าแอดมินที่ใช้แบ่งออกเป็นหน้าของ Admin Account Page ที่ใช้สำหรับจัดการข้อมูลของ Admin และ Admin Product Page สำหรับการดูแลสินค้าต่างๆ
6.1. Admin Account Page
- โดยจะใช้ในการดูแล User และแก้ไขข้อมูลของ User เช่น การเพิ่มข้อมูล ,  การแก้ไขข้อมูล และ การลบข้อมูล
6.2 Admin Product Page
- จะใช้สำหรับการดูแลข้อมูล Product และใช้ในการแก้ไขข้อมูลของ Product ได้แก่ การเพิ่มข้อมูล , การแก้ไขข้อมูล และ การลบข้อมูล

*** ทุกหน้าเป็นการจำลองเพจของ Website เท่านั้นจะไม่มีการทำงานด้านการประมวลผลจริงของเว็ปไซต์(Fixed Page) ***
